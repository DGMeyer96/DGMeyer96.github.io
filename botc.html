About

This project was created for the CSUSB CSE Game Design class and competition. 11 teams were given 10 weeks to design and develop a game of their choice. On the 10th week each of the teams demonstrated the games in front of a panel of judges and a winner was decided. The game's design was of my creation and I served as Project Lead for this project as well as Programmer and 3D Artist.

Development

Developement was done in Unity using the Lightweight Render Pipeline, Cinemachine, Shader Graph, and VFX Graph.

As Project Lead I was in charge of managing the GitHub Repository and weekly SCRUM meetings among the programmers. During each meeting the programmers would update me on their progress and I would assign new weekly goals as the project progressed.

As 3D Artist I created the majority of the 3D models for the game with the exception of the player model which was provided by a fellow student at CSUSB, JC Mariscal. The models were created using Cinema4D with texture work done in Adobe Photoshop.

As Programmer I worked on the game's back-end, audio, and graphics. For the back-end work I managed the project settings, programmed the game's UI, menus, settings menu, loading and scene transitions, as well as some work on the combat system. In the audio department I created the BGM and SFX playback with audio transitions for the battle and exploration track. For the game's graphics I setup the post-processing pipeline and lighting model. The game used a procedural sky through Unity's Post-Processing pipeline. Finally the game also features cel-shading and a cartoon outline with a posterizing effect through a custom render pipeline asset as well as custom shaders developed through Unity's Shader Graph.
